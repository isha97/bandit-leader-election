# leader-election-bft

## Log Tags
* `[SEND] - Message Sent`
* `[RECV] - Received Message`
* `[Leader] - Leader ID status`
* `[LeaderElec] - Leader election`
* `[FailEst] - Failure Estimates`
* `[FailEst (UP/DOWN)] - Failure Estimates update`
* `[Status] - Node Status`
* `[Client] - Messages to/from Client`
